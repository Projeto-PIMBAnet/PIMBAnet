# PIMBAnet

Nosso projeto é um sistema de monitoramento de rede, com a função de registrar dispositivos e verificar a conectividade de vários aparelhos localizados em uma única rede.

---

## Requisitos

Faça o download do banco de dados, implemente o script no seu MySQL e execute-o utilizando o XAMPP.

---

## Funcionalidades

- Registrar um dispositivo  
- Testar conectividade  
- Verificar histórico de dispositivos  
- Consultar logs de testes  

---

## Estrutura do Projeto

server/PIMBAnetCore/src  
│── styles  
│── App.jsx  
│── Historico.jsx  
└── main.jsx  

---

## Tecnologias Utilizadas

- React  
- CSS  
- JavaScript  
- MySQL  

---

## Como executar

1. Clone o repositório  
2. Acesse as pastas **PIMBAnetCore** e **PIMBAnetBack** separadamente  
3. Inicie o servidor MySQL via XAMPP  
4. No terminal do **PIMBAnetBack**, execute: node server.js
5. No terminal do **PIMBAnetCore**, execute: npm run dev
6. Acesse o link gerado pela execução
